# RailsJobs

Versions
------------------------------------
- Rails 5
- ruby 2.3.1


# Enviroment setup
* install postgress and set the user "postgres" and password="password"
* docker-compose build
* docker-compose run --rm app rake db:create db:migrate


## Running the app
* docker-compose up

## stoping the app
* docker-compose stop or 'ctrl+c'
